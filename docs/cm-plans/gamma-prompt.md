# Gamma で提案書を作る（無料枠でOK）

Gamma（gamma.app）は登録時の無料クレジットで1本は生成できます。以下の手順＋アウトラインで、シネマ画像つきの提案スライドができます。

## 手順（3分）
1. Gamma → **Create new → Presentation →「Paste in text（テキストを貼り付け）」**
2. 下の【貼り付け用アウトライン】を全部貼る。カード区切りは `---` で認識されます
3. 生成前オプション：
   - **Amount of text per card**：Detailed（しっかりめ）
   - **Image style**：下の「画像スタイル」をコピペ
   - **Theme**：ミニマルで上品なもの（例：Oasis / Consult / Founder など明朝系・余白広め）
4. **Generate** → 生成後、各カードの画像が弱ければ「画像を再生成」＋各カードに記した英語プロンプトを貼る

## 画像スタイル（Image style 欄に貼る）
```
cinematic, photorealistic, natural lighting, elegant Japanese aesthetic, muted washi and green palette, no text, no logos, no flags, 16:9
```

## クレジット節約のコツ
- まずは**章扉＋主要カードだけ画像あり**で生成（全カード画像だと消費大）
- 気に入ったら個別に画像追加。テキストは何度直しても軽い
