# CM品質を上げる 外部ツール活用ガイド（プロンプト／差し替えだけ）

「プロンプトを貼るだけ」「生成物を差し替えるだけ」で完結する、手間の少ない品質アップ手順。
制作プロンプトは `docs/cm-plans/green-battery-production.md` に全カット分あります。

---

## いちばん簡単な全体フロー（3ステップ）
1. **静止画をつくる** … production.md のDALL·E用プロンプトを画像AIに貼る → 各カットのキーフレーム
2. **動画にする** … その画像を動画AIに入れ、Kling用プロンプト（動きの指定）を貼る → 各カット5秒
3. **提案書に反映** … 生成した静止画を私（Claude）に渡すだけ → 絵コンテを実写フレームに差し替え

> ※ 提案書(Artifact)はセキュリティ上、外部URLの画像を読み込めません。**画像ファイルをこのリポジトリに置いてもらえれば、私が埋め込み(data URI化)して差し替えます。** = 御社は「画像を置くだけ」。

## 用途別・おすすめツール（すべてプロンプト入力が中心）

### 静止画（キーフレーム）
| ツール | 特徴 | 手軽さ |
|---|---|---|
| **Midjourney** | 実写・シネマ品質が最高峰。`--ar 16:9` で横長指定 | Discordにプロンプト |
| **DALL·E 3（ChatGPT）** | 日本語プロンプトOK・最も手軽 | チャットに貼るだけ |
| **Adobe Firefly** | 商用利用の権利がクリアで安心 | サイトに貼るだけ |
| **Google ImageFX / Imagen** | 自然な風景に強い | プロンプトのみ |

### 動画（静止画→動画）
| ツール | 特徴 |
|---|---|
| **Kling**（現行） | 画像から動画。人物の動きが自然 |
| **Runway Gen-3** | カメラワーク指定が得意（push in 等） |
| **Luma Dream Machine** | なめらか・無料枠あり |
| **Google Veo / Pika** | 高品質・用途で使い分け |

### 音楽・ナレーション・仕上げ
| 用途 | ツール | 注意 |
|---|---|---|
| BGM（和風も可） | **Suno / Udio** | 生成音楽の商用ライセンスを必ず確認。心配ならEpidemic Sound等の権利クリア音源 |
| 英語ナレーション | **ElevenLabs** | テキスト入力だけで米国ネイティブ音声 |
| 編集・テロップ | **CapCut** | テンプレ＋自動字幕。ロゴ・コピーはここで合成 |
| 提案スライド化 | **Gamma** | プロンプトでスライド生成（本HTMLの補助に） |

## 品質を上げる小ワザ（プロンプトに足すだけ）
- 実写感：末尾に `shot on ARRI Alexa, 35mm, cinematic color grading, shallow depth of field`
- 光の質：`golden hour, volumetric light, soft rim light`
- 画角の統一：全カット同じ `--ar 16:9`（縦型は `--ar 9:16`、被写体を中央に）
- 一貫性：人物・施設は「1枚の承認済みマスター画像」を参照に固定（Midjourneyは `--cref`／`--sref`）
- 文字は入れない：`no text, no watermark, no logo`（ロゴ・コピーは編集で）

## やってはいけない（品質より信頼のため）
- 生成音楽・素材の**商用ライセンス未確認のまま有料配信（Netflix）に使う**
- 実在人物・実在ブランド・旭日旗の混入（プロンプトに `no real brands, no flags`）
- 未確認の数値・実績をテロップ／ナレーションに入れる

## 手持ちの有料ツールの活かし方（HeyGen / Kling）

### 役割分担（重要）
- **Kling AI＝CMの映像本体**（富士・桜・歌舞伎・施設など）。画像→動画で各カットを生成。
- **HeyGen＝声と多言語、そして"人が話す"動画**。CMのシネマ映像は作れないので、下記に使う。

### Kling（映像）の使いこなし
- **画質**：高品質モード／1080pで出力
- **画像→動画**：`production.md` のDALL·E静止画を「開始フレーム」にすると一貫性が安定（可能なら終了フレームも指定してカメラの動きを制御）
- **モーションブラシ**：桜の花びら・水・光など「動かしたい箇所」だけ指定
- **カメラ制御**：`production.md` の各カットのカメラ指定（空撮／ドリー／スナップズーム等）に対応させる
- **複数テイク**：1カットにつき数回生成し、一番良いテイクを採用（5秒生成→編集で2〜3秒に詰める）
- **ネガティブ**：`text, watermark, logo, distortion, extra fingers`

### HeyGen（声・多言語・スポークスパーソン）
- **英語ナレーション**：AI Voice（米国ネイティブ）で各案のVOを生成 → ElevenLabs不要。トーンは「静かで信頼感のある低〜中音」。台本は各案のVO行（`green-battery-proposal.html`／`production.md`）。
- **多言語版**：Video Translate で **日本語版（国内・投資家用）** と **英語版（Netflix用）** を出し分け。
- **投資家・M&A用の拡張版**：ビジネス装のAIアバターに **30〜60秒のスポークスパーソン動画**を語らせる（会社紹介＋"Partner with us"）。※これは"人が話す"動画なのでHeyGenの得意分野。CM本編には使わない。
- **提案時の一工夫**：提案PDFに、HeyGenアバターが4案を1分で紹介する動画を添えると商談での掴みが強い（任意）。

### 静止画（Klingの開始フレーム）の作り方 ※無料でOK
Kling自体に画像生成があればそれを、無ければ **DALL·E 3（無料枠）／Adobe Firefly（商用安全）** で `production.md` のプロンプトから生成 → Klingへ。

### まとめ：最短ワークフロー
1. 静止画：DALL·E/Firefly（無料）で各カット生成
2. 動画：**Kling**（有料）で画像→動画、各カット5秒
3. 声：**HeyGen**（有料）で英語VO＋（必要なら）日本語版
4. 編集：CapCut（無料）で連結＋ロゴ・テロップ合成 → 1080p書き出し
5. 投資家用：**HeyGen**アバターで30〜60秒スポークスパーソン版
