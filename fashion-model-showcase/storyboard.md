# AIモデル生成サンプル動画「あなたのブランドに、最適なモデルを。」篇 — 絵コンテ仕様書

**コンセプト：** 同じ背景・衣装でモデルだけが切り替わることで「AIで自由にモデルを生成・差し替えできる」を直感的に伝える
**尺：** 30秒
**ナレーション：** あり
**BGM：** あり（スタイリッシュ・モダン）
**テキスト：** 最小限

## 詳細台本

### S1 — 日本人女性モデル（0〜6秒）

- 白いスタジオ。ソフトな均一照明。
- 20代・日本人女性モデルがシンプルなホワイトドレスを着て、カメラに向かってゆっくり歩いてくる。
- 表情は自信に満ちた柔らかい微笑み。
- カメラは固定。モデルが近づくにつれてフォーカスが引き締まる。

**画面テキスト：** なし

---

### S2 — 欧米系女性モデルに切り替わり（6〜12秒）

- 背景・衣装・照明・カメラアングルはS1と完全に同じ。
- モデルだけが30代・欧米系女性に変わっている。
- 同じように歩いてくる。視聴者が「あれ、変わった？」と気づく瞬間を演出。

**画面テキスト：** なし（気づかせるため、説明テキストはまだ出さない）

---

### S3 — さらに切り替わり（12〜18秒）

- 40代・アジア系女性（3秒）→ 20代・男性モデル（3秒）と素早く切り替わる。
- いずれも同じ背景・衣装（男性モデルのみスーツに変更可）。
- テンポが上がり、「どんどん切り替えられる」ことへの驚きが増していく。

**画面テキスト：** なし

---

### S4 — 種明かし（18〜24秒）

- 画面が暗転し、「モデルも映像も、すべてAIで生成しました。」テキストがフェードイン。
- 背景に光の粒子・生成感のあるビジュアルエフェクト（CapCutのテンプレートで代用可）。
- 下部に小さく「AI Model Generation / AI Video Production」と表示。

**画面テキスト：**
```
モデルも映像も、すべてAIで生成しました。
AI Model Generation / AI Video Production
```

---

### S5 — エンドカード（24〜30秒）

- ロゴフェードイン。
- キャッチコピー表示。

**画面テキスト：**
```
あなたのブランドに、最適なモデルを。
[会社名・URL]
```
